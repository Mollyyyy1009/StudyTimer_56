# StudyTimer 学习辅助与计划系统

StudyTimer是一个集 **记忆曲线回顾、日程规划、学习数据跟踪、番茄计时器** 于一体的学习辅助项目。  
项目目标是帮助用户提升学习效率、优化计划安排并形成持续的学习节奏。

---

## ✨ 功能简介

### 1. 记忆曲线回顾
基于艾宾浩斯记忆曲线自动生成复习节点。

### 2. 日程规划
自动调度任务，支持优先级、标签过滤、时间片分配等功能。

### 3. 番茄计时器
标准 25 分钟番茄工作法，可记录每次计时与任务完成情况。

### 4. 数据统计
学习时长统计、任务完成量图表、学习标签分布可视化。
